WakiVo – Account Deletion Request

If you would like to delete your WakiVo account, you can submit a deletion request using the email below:

support@yourapp.com

Please make sure to send your request from the same email address used to create your account so we can verify ownership.

Data That Will Be Deleted

  • User account
  • Email address
  • Password
  • Voice messages and audio files
  • Profile data

Data That May Be Temporarily Retained

For security or legal purposes, the following information may be retained for up to 30 days before being permanently removed:

  • Security logs
  • System logs
  • Information required to prevent fraud or misuse

Processing Time

Your deletion request will be processed within 7–30 days. You will receive a confirmation once the deletion is complete.
